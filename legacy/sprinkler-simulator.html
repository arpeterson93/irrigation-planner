How this works

Precipitation rate

Each head's precipitation rate is 96.3 × GPM ÷ coverage area (sq ft), giving inches/hour. Coverage area is a circular sector: π × radius² × (arc span ÷ 360). This is the standard irrigation-industry formula and is what the rest of the math is built on.

Heatmap

For every grid cell, we check which heads reach it (distance ≤ radius, and bearing to the cell falls within the head's arc). Each contributing head adds its precipitation rate; multiplying by that head's zone run time gives inches applied in one cycle. A cell's total is the sum across all zones' single cycle.

Forecast ET₀ estimate

Reference evapotranspiration is estimated with the Hargreaves-Samani equation using your latitude and the NWS-forecasted daily high/low temperature — a widely used simplified method when a dedicated ET station isn't available. It's an estimate, not a substitute for a local CIMIS/AgriMet reading if you have access to one — use the override field if so.

Seasonal adjustment

Net water need = ET₀ − (forecasted rain × runoff efficiency). Adjustment % = net need ÷ baseline daily need. Each zone's suggested run time = its configured run time × adjustment % (clamped 0–150%).

Your data

Everything you enter is saved automatically in this browser (localStorage) — nothing is sent anywhere except the two anonymous, keyless calls to the public NWS API when you fetch a forecast. Use Export JSON any time to back up or move your yard to another device, and Import JSON to restore it.
